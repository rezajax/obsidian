### 📚 ساختار گذشته:

برای صرف گذشته، از ریشه‌ی فعل استفاده شده و پسوندهای خاص افزوده می‌شود.

|   |   |   |
|---|---|---|
|مصدر|معنی|گذشته من (اول‌شخص مفرد)|
|ուտել|خوردن|կերա (kerá)|
|խմել|نوشیدن|խմեցի (khmetsi)|

### 🧩 جمله‌های نمونه:

- Նա եկավ։ = او آمد.
    
- Մենք կարդացինք։ = ما خواندیم.
